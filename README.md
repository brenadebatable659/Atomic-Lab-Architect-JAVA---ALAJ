## 🔬 Main features

### ⚛️ Electron Configuration

ALAJ can generate electron configuration in subshell notation. This helps you see how electrons fill orbitals in a clear format.

### 🧬 VSEPR Geometry

The app uses VSEPR theory to predict molecular shape. It can help you understand whether a molecule is linear, bent, trigonal planar, tetrahedral, or another common form.

### 🌡️ Physical State Prediction

ALAJ estimates the physical state of a substance. It uses basic chemistry patterns to suggest if a substance may be solid, liquid, or gas at room temperature.

### 🧊 Boiling Point Estimate

The app estimates boiling points from chemical data and related trends. This can help you compare substances and study how bonding affects behavior.

### 🌀 Molecular Dynamics Simulation

ALAJ includes a Lennard-Jones style simulation. It gives a simple view of how molecules may interact and stabilize over time.
