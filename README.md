# Atomic-Lab-Architect-JAVA---ALAJ
ALAJ : A Java-based Digital Chemistry Lab, chemical analysis tool and molecular dynamics simulator.Features:1.Electron Configuration: Automatically generates subshell notation.2.VSEPR Theory: Predicts molecular geometry.3.MD Simulation: Runs a Lennard-Jones simulation to stabilize molecules.4.Physical State Prediction.5.Estimates boiling points,phases.
